# PHTS LVL-1

![](./doc/model.png)

Features:

- Use ultrasonic sensor to measure distance
- Show level on the big fancy indicator
- Send levels and logs to [szambo.tsaryk.com] via WiFi
- 24/7 monitoring

## Firmware

Consists of 2 modules: [Arduino Nano](./arduino) and [NodeMCU V3](./nodemcu).
